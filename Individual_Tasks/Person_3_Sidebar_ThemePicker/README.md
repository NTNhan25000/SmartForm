# Person 3 - Sidebar & Theme Picker

## 🎯 Mục tiêu
Phụ trách công cụ hỗ trợ: sidebar thêm câu hỏi nhanh và theme picker trực quan.

## 📋 Tiêu chí hoàn thành
- [x] Sidebar hiển thị 8 loại câu hỏi rõ ràng
- [x] Theme picker có hệ phân loại (Màu đơn vs Gradient)
- [x] Theme selected rõ ràng
- [x] Có preview của theme đang chọn
- [x] Icon + label dễ hiểu
- [x] Responsive ổn trên mobile

## 📁 Cấu trúc file
- `index.html` - HTML structure
- `style.css` - CSS styling
- `script.js` - JavaScript functionality

## 🚀 Hướng dẫn sử dụng
1. Mở `index.html` trong trình duyệt
2. Click vào các loại câu hỏi để thêm
3. Click vào màu theme để chọn
4. Xem preview cập nhật

## 🎨 Loại câu hỏi hỗ trợ
- 🅰️ Trả lời ngắn
- 📝 Đoạn văn
- ⭕ Lựa chọn nhiều
- ☑️ Hộp kiểm
