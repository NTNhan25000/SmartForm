# Person 5 - AI Chatbox & Responsive Polish

## 🎯 Mục tiêu
Phụ trách điểm khác biệt: AI Chatbox hiện đại và responsive polish cho toàn bộ giao diện.

## 📋 Tiêu chí hoàn thành
- [x] AI chatbox không quá lớn, nổi bật đúng mức
- [x] Trạng thái online/offline/checking rõ ràng
- [x] Quick action chips dễ sử dụng
- [x] Message bubbles phân biệt AI vs User
- [x] Responsive hoạt động tốt trên mobile
- [x] Toast notifications hiển thị đúng
- [x] Chatbox mở/đóng mượt mà

## 📁 Cấu trúc file
- `index.html` - HTML structure
- `style.css` - CSS styling
- `script.js` - JavaScript functionality

## 🚀 Hướng dẫn sử dụng
1. Mở `index.html` trong trình duyệt
2. Click nút AI (✨) ở góc phải để mở chatbox
3. Chọn quick action hoặc nhập câu hỏi
4. Xem AI trả lời
5. Test responsive bằng F12

## ⚡ Features

### AI Chatbox
- Toggle button floating
- Header với status
- Message history
- Typing indicator
- Quick action buttons
- Input area với send button

### Status Indicators
- 🟢 Online (Sẵn sàng)
- 🔴 Offline (Ngoại tuyến)
- 🟡 Checking (Đang kiểm tra)

### Quick Actions
- 💡 Gợi ý câu hỏi
- ✏️ Cải thiện tiêu đề
- ➕ Thêm câu hỏi

### Responsive
- Desktop: Sidebar chatbox (360px)
- Mobile: Fullscreen chatbox (100vw, 70vh)
- Tablet: Adjustable size

## 🎨 Design Language
- Primary Gradient: #667eea → #764ba2
- Light Background: #f5f5f5
- AI Message: #f0f0f0
- User Message: #667eea
