# Person 2 - Form Header & Question Card

## 🎯 Mục tiêu
Phụ trách phần cốt lõi: tiêu đề, mô tả form và các question card đang được chỉnh sửa.

## 📋 Tiêu chí hoàn thành
- [x] Header form có hierarchy rõ (tiêu đề nổi bật)
- [x] Card selected nhìn khác biệt rõ với card thường
- [x] Hover, focus, disabled, required rõ ràng
- [x] Menu thao tác không che nội dung
- [x] Các loại câu hỏi hiển thị đúng
- [x] Thao tác nhanh: copy, delete, add

## 📁 Cấu trúc file
- `index.html` - HTML structure
- `style.css` - CSS styling
- `script.js` - JavaScript functionality

## 🚀 Hướng dẫn sử dụng
1. Mở `index.html` trong trình duyệt
2. Click vào câu hỏi để chọn
3. Chỉnh sửa câu hỏi, tiêu đề, mô tả
4. Sử dụng nút Copy/Delete
5. Click "Thêm câu hỏi" để thêm mới

## 🎨 Design Language
- Primary Color: #4285f4
- Active State: #f0f4ff (light blue background)
- Border: #e0e0e0
- Type Badge: #f5f5f5

## ⚡ Features
- Form header (title + description)
- Question cards với 3 ví dụ
- Card selection & highlighting
- Copy & Delete functionality
- Add new question button
- Question type badge display

## 📝 Ghi chú
- Sidebar (Person 3) sẽ hỗ trợ thêm các loại câu hỏi
- Top bar (Person 1) sẽ xử lý trạng thái lưu
